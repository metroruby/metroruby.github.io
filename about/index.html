Story

จุดเริ่มต้นของผม ย้อนไปเมื่อ 10 กว่าปีที่แล้ว ผมได้คอมพิวเตอร์เครื่องแรกในชีวิต ตอนนั้นผมก็คิดแต่ว่ามีคอมเท่ากับมีเกมให้เล่นมากมาย ซึ่งมันก็เป็นความจริงครับระยะเวลานั้น ผมสนใจแต่เกมช่วงนั้น Counter strike 1.6 กำลังมาแรง ซึ่งผมก็ไม่พลาดครับ เก็บตังค์เพื่อที่จะซื้อเกมมาเล่น มีเกมอะไรมาใหม่ผมก็จัดการซื้อมาลงอย่างต่อเนื่อง จนอยู่มาวันหนึ่งคอมผมได้เกิดอาการเปิดเครื่องไม่ติด ผมก็คิดทำยังไงดีเกมก็เล่นไม่ได้จะไปบอกพ่อก็กลัวจะโดนว่า ซึ่งตอนนั้นผมไม่มีความรู้อะไรเลยนอกจากเกม สุดท้ายผมก็ต้องพึ่งพ่อครับ พ่อผมก็ต้องยกคอมไปซ่อมที่ร้าน ซึ่งมันก็ต้องเสียค่าใช้จ่าย ในตอนนั้นไปซ่อมก็เสียครั้งละ 500 บาท มันเป็นแบบนี้มาตลอดครับ จนมาถึงตอนที่ผมคิดว่า ทำไมเราต้องไปเสียตังค์ให้ร้านซ่อมด้วยละ นั้นคือสิ่งที่จุดประกายความคิดของผมว่า ทำไมเราไม่เอาเงินที่เก็บซื้อเกมไปซื้อแผ่น Windows มาลงเองสะเลยละ ผมก็ไม่รอช้าครับจัดการซื้อแผ่น Windows มาเรียบร้อย ตอนนั้นล่าสุดคือ Windows Me ครับ ตัวผมเองไม่รู้เลยว่าต้องทำยังไง รู้อย่างเดียวว่ายัดแผ่นเข้าเครื่องแล้วกดลงมันสะ เรียบร้อยครับอย่างที่คิดไว้ เจ๊ง!!! ครับ ลงไม่สำเร็จ แต่นั้นก็คือจุดเริ่มต้นในการหาข้อมูลต่างๆ นาๆ ของผมว่าต้องทำยังไงถึงจะลงได้ ต้องแก้ปัญหายังไง

หลังจากนั้นผมคิดแต่ว่าผมต้องเข้าเรียนสายคอมพิวเตอร์ให้ได้ จนถึงเวลาจบ ม.6 ผมก็ได้เข้าสมใจครับสายคอมพิวเตอร์อย่างที่ผมฝันเอาไว้ ผมได้เรียนต่อที่ มหาลัยเทคโนโลยีพระนครเหนือ วิทยาเขตปราจีนบุรี สาขา IT ที่ผมต้องการ หลังจากนั้นความคิดด้านคอมพิวเตอร์ของผมก็ได้เปลี่ยนไปครับ เปลี่ยนไปยังไงก็คือว่าผมก็ยังชอบด้านคอมเหมือนเดิมและครับ แต่ผมได้รู้เพิ่มขึ้นมาว่าด้านนี้มันมีสาขาแยกอีกมากมาย นั้นทำให้ผมต้องคิดว่า ผมจะเลือกไปเรียนเอกอะไรนั้นเองครับและมันก็จำเป็นมากครับ เพราะถึงเวลา ปี3เทอม 1 ผมต้องเลือกแล้วครับว่าจะเรียนเอกอะไร มันมีให้เลือกทั้งหมด 4 เอกครับ โปรแกรมเมอร์ ระบบเครือข่าย มัลติ ธุรกิจสุดท้ายผมได้เลือก มัลติครับ ทำไมตอนนั้นถึงเลือกมัลติ ย้อนกลับไปเมื่อตอนเด็กครับที่ผมเล่าให้ฟังว่าผมชอบเล่นเกม นั้นและครับเอกนี้มันมีการสร้างเกมครับ ผมเลยเลือกเอกมัลติครับ หลังจากเรียนจบ ผมได้นั่งถามตัวเองว่าผมชอบวิชาอะไรอีกบ้างสุดท้ายผมได้ตอบคำถามตัวเองได้ครับ นั้นก็คือ ผมอยากรู้เรื่อง Network เพิ่มเข้ามาอีกจึงทำให้ผมต้องหาข้อมูลมากมายเกี่ยวกับ Network หาอ่าน และทดลองทำเอง

ต่อมา ผมได้ทำงานในด้าน System and Network and Admin ให้กับบริษัท Transport แห่งหนึ่ง เนื้องานจะเริ่มตั้งแต่ติดตั้งระบบไปจนถึงคอยดูแล User ในเรื่องต่าง ๆ คอย Monitor ตรวจสอบความผิดพลาดของระบบ และได้กำลังศึกษา Linux, Security เพิ่มเติมเพื่อจะทำการทดสอบทำ System ขึ้นมารองรับ User ครับ

ปัจจุบัน ผมได้ทำงานในสายงาน System Engineer ให้กับบริษัท SI แห่งหนึ่ง เนื้องานคือ Implement ระบบให้กับองกรณ์เช่น ทำ Private Cloud, Virtualization, Software Defined, Database ORACLE MsSQL, Environment for APP, Software Backup, VDI, Desktop Management, migrate app to cloud, และออกแบบ Topology network เบี้ยงต้น
